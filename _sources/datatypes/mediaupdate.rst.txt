===========
MediaUpdate
===========

.. |br| raw:: html

    <br />


.. role:: dt
   :class: datatype


Summary
-------

The MediuUpdate model is received on requesting an update of metadata on a media
object.


Model
-----

.. code-block:: Javascript

   {
      "author": "",
      "title": ""
   }


Detail
------

**author**
   New author name.

   **Type**: :dt:`String` |br|
   **Default Value**: ``""``
   

**title**
   New title name.
   
   **Type**: :dt:`String` |br|
   **Default Value**: ``""``
