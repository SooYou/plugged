=======
ModSkip
=======

.. |br| raw:: html

    <br />


.. role:: dt
   :class: datatype


Summary
-------

Model emitted when a mod skips the current play.


Model
-----

.. code-block:: Javascript

   {
      "moderator": "",
      "moderatorID": -1
   }


Detail
------

**moderator**
   Name of the moderator.

   **Type**: :dt:`String` |br|
   **Default Value**: ``""``
   

**moderatorID**
   ID of the moderator.
   
   **Type**: :dt:`Number` |br|
   **Default Value**: ``-1``
