================
BOOTHBANDURATION
================

Summary
-------

The BOOTHBANDURATION enumeration is, as the name implies, used for the duration a
user should be banned for from the booth.

.. note::
   A quarter means 15 minutes


Enum
----

   * 's': QUARTER
   * 'm': HOUR
   * 'l': DAY
   * 'f': PERMANENT
