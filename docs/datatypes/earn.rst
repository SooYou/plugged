====
Earn
====

.. |br| raw:: html

    <br />


.. role:: dt
   :class: datatype


Summary
-------

Model emitted when you earn experience.


Model
-----

.. code-block:: Javascript

   {
      "xp": -1,
      "pp": -1,
      "level": -1
   }


Detail
------

**xp**
   Amount of XP gathered.

   **Type**: :dt:`Number` |br|
   **Default Value**: ``-1``
   

**pp**
   Amount of Plug points in your wallet.
   
   **Type**: :dt:`Number` |br|
   **Default Value**: ``-1``


**level**
   Your current level.
   
   **Type**: :dt:`Number` |br|
   **Default Value**: ``-1``
