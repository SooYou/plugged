=========
Inventory
=========

.. |br| raw:: html

    <br />


.. role:: dt
   :class: datatype


Summary
-------

This model represents an item you own.


Model
-----

.. code-block:: Javascript

   {
      "category": "",
      "id": "",
      "type": ""
   }


Detail
------

**category**
   Category of the item, i.e. "classic".
   
   **Type**: :dt:`String` |br|
   **Default Value**: ``""``


**id**
   Unique identifier of the item.
   
   **Type**: :dt:`String` |br|
   **Default Value**: ``""``


**type**
   Type of the item, i.e. "avatars".

   **Type**: :dt:`String` |br|
   **Default Value**: ``""``
