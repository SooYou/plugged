====
Vote
====

.. |br| raw:: html

    <br />


.. role:: dt
   :class: datatype


Summary
-------

Model emitted when someone votes.


Model
-----

.. code-block:: Javascript

   {
      "avatarID": "",
      "badge": "",
      "id": -1,
      "level": -1,
      "username": ""
   }


Detail
------

**avatarID**
   Avatar used.

   **Type**: :dt:`String` |br|
   **Default Value**: ``""``
   

**badge**
   Badge used.

   **Type**: :dt:`String` |br|
   **Default Value**: ``""``
   

**id**
   ID of the user.
   
   **Type**: :dt:`Number` |br|
   **Default Value**: ``-1``


**level**
   Current level.

   **Type**: :dt:`Number` |br|
   **Default Value**: ``-1``
   

**username**

   **Type**: :dt:`String` |br|
   **Default Value**: ``""``
